# 🚨 Phishing URL Detection System

## 📌 Project Overview
This project is a Machine Learning based phishing website detection system that predicts whether a website is legitimate or phishing using cybersecurity-related URL features.

## 🔥 Features
- Machine Learning based phishing detection
- Random Forest Classifier
- Streamlit Web Application
- Real-time prediction interface
- Reset button support
- User-friendly UI

## 🛠 Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- Joblib
- Matplotlib
- Seaborn

## 🚀 How to Run

### 1️⃣ Clone Repository

```bash
git clone https://github.com/srijabojja-07/phishing-url-detection-system.git
```

### 2️⃣ Install Requirements

```bash
pip install -r requirements.txt
```

### 3️⃣ Run Streamlit App

```bash
cd phishing-url-detection-system
cd app
streamlit run app.py
```

## 📊 Machine Learning Model
- Random Forest Classifier
- Trained on phishing website dataset
- Used cybersecurity URL features for classification

## 🎯 Output
- ✅ Legitimate Website
- ⚠️ Phishing Website Detected
